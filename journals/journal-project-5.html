My Canva — Web-based Task Management Tool

About the Project

My Canva is a collaborative web-based task management tool inspired by Trello, built with Angular, Spring Boot, and MySQL. It allows users to create customizable canva boards, define phases, and manage tasks through an intuitive drag-and-drop interface. Designed for teamwork, My Canva supports user roles and project invitations, making it ideal for organizing workflows in both academic and professional settings.

The project was developed as part of the Advanced Web Programming course, serving as hands-on experience in building scalable, interactive full-stack applications.
