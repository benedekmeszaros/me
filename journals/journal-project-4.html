Peek — Gallery Web Application

About the Project

Peek is a full-stack social media gallery web application, inspired by platforms like Reddit. Built with React, Spring Boot, and MySQL, the app allows users to create and contribute to categorized image galleries, engage with posts through voting and comments, and follow other users for real-time content updates. Designed as a dynamic, modern platform, Peek brings visual communities together through simple interactions and seamless navigation.

This project was developed as part of a university course on Development of Informational Systems, showcasing a comprehensive understanding of full-stack architecture and modern web technologies.

Key Features

  • Gallery-Based Content: Users can create and browse categorized galleries
  • Post and Interact: Share images, upvote/downvote posts, and leave comments.
  • User Connections: Follow users to stay updated with their latest posts.
  • Notification System: In-app alerts notify users when someone they follow posts new content.
  • JWT Authentication: Secure login and authorization powered by JSON Web Tokens.
  • Tech Stack:
    • Frontend: React
    • Backend: Spring Boot
    • Database: MySQL

Post Thoughts

Peek was a valuable learning experience in building and connecting a robust frontend and backend system. It not only demonstrated how to structure a modern full-stack application but also highlighted the importance of clean architecture, authentication, and user interaction features. While originally created for academic purposes, the project has the potential for further development and refinement into a production-ready platform.
